## 🏗️ Tech Stack

**Backend:** Node.js, Express.js, TypeScript, Supabase (PostgreSQL)  
**Frontend:** React 19, Vite 7, TypeScript, shadcn/ui, Tailwind CSS  
**Deployment:** Vercel (serverless)

## ⭐ Key Features

- Multi-tenant architecture (Student, Admin, Driver portals)
- Google OAuth authentication
- Payment gateway integration (Razorpay)
- **Optimization algorithm** (constraint programming)
- QR-based validation system
- Comprehensive admin analytics
- Email notifications & automated reminders
- Financial reporting with history
- Mobile-responsive design

## 🚀 Getting Started

### Backend
```bash
cd backend
npm install
cp .env.example .env
# Edit .env with your configuration
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

For detailed setup instructions, see the README files in `/backend` and `/frontend` directories.
